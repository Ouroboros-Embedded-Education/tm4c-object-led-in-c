## Requisitos

Instale o **TivaWare for TM4C SDK 2.2.0.295** para acesso ao Driverlib da linha TM4C da Texas, e verifique se está instalado o Compilador **TI v20.2.7LTS**.

Você pode verificar estas informações em *File->Preferences->Code Composer Studio Settings-> General*, a SDK fica na aba **Products** e o compilador em **Compilers**.

A SDK pode ser instalada pelo **Resource Explorer**, no menu **View**.
